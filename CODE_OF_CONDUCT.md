# Code of Conduct

## Our Standard

This project is maintained as a public portfolio and educational repository. Discussions should stay practical, respectful, and focused on improving the project.

Examples of good behavior:

- clear issue reports;
- constructive pull request feedback;
- respectful technical disagreement;
- safe examples without private data;
- useful documentation improvements.

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or personal attacks;
- publishing secrets or private data;
- spam;
- abusive language;
- irrelevant promotion;
- requests to bypass legal or platform restrictions.

## Enforcement

Maintainers may edit, close, delete, or lock issues and pull requests that violate this code of conduct.

## Reporting

Report problems through GitHub issues with a safe, factual description. Do not include private data, credentials, API keys, or personal information.
